# Tax Audit Beta Pilot Checklist

Beta label: Tax Audit Beta v0.1 RC1  
Scope: AY 2025-26, Rule 6G, Forms 3CA, 3CB and 3CD  
Phase: Phase 1 - Structured Tax Audit Workflow

## Pilot Data Safety

Pilot users should avoid using confidential client data unless specifically permitted by ICAI or their firm. For pilot testing, use dummy data or anonymised data where possible.

## Functional Smoke Checklist

- [ ] Fresh install opens successfully.
- [ ] Existing local data opens successfully, if available.
- [ ] Create or open a tax audit engagement.
- [ ] Complete setup and applicability.
- [ ] Complete acceptance and eligibility checklist.
- [ ] Complete compliance tracker.
- [ ] Complete 3 to 5 representative Form 3CD clauses.
- [ ] Add, edit and delete rows in structured clause tables.
- [ ] Use Audit Programme and save at least one checklist item.
- [ ] Acknowledge Professional Responsibility and Disclaimer.
- [ ] Check Report Readiness status and attention items.
- [ ] Generate Working Paper Report Pack preview.
- [ ] Copy Working Paper Report Pack summary.
- [ ] Download Working Paper Report Pack Markdown.
- [ ] Close and reopen the app.
- [ ] Verify setup, clause response, Audit Programme item, acknowledgement and report pack source data remain available after reopening.

## Suggested Clause Coverage

- [ ] One basic identity/setup clause, such as Clause 1 to Clause 8.
- [ ] One books/accounting clause, such as Clause 11 to Clause 14.
- [ ] One table-heavy clause, such as Clause 21, Clause 26, Clause 34 or Clause 44.
- [ ] One review item with internal remarks.
- [ ] One item with qualification or observation wording, if relevant to the dummy case.

## Feedback Capture Guidance

For every issue, tester should capture:

- Firm name.
- Tester name.
- Date.
- Operating system.
- App version.
- Tax Audit module version.
- Module/tab.
- Steps to reproduce.
- Expected result.
- Actual result.
- Screenshot, if possible.
- Severity: blocker / major / minor / suggestion.

## Known Beta Limitations

- Final statutory Form 3CD generation is not part of this beta.
- Portal filing is not part of this beta.
- PDF, Word and Excel export are not part of this beta.
- GST, TDS and purchase register imports are not part of this beta.
- Structured clause capture is manual and requires professional review.
- Working Paper Report Pack is an internal working paper summary, not a signed report.

## Pilot Completion Sign-off

- [ ] Pilot tester confirms core workflow can be completed with dummy or anonymised data.
- [ ] Pilot tester confirms saved data persists after app restart.
- [ ] Pilot tester confirms known limitations are clear.
- [ ] Pilot tester confirms blockers, if any, are reported with reproduction details.
